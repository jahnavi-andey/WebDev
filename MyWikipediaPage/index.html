Major Projects

Jahnavi has engaged in various projects that exemplify her proficiency in web development, data analysis, and application development. The following projects highlight her contributions in these fields:

EmoNarrate

EmoNarrate is an application developed to convert text from books into audio, significantly improving accessibility for users, especially for visually impaired individuals, who comprise approximately 10% of the user base. The application features user-friendly browsing and book selection functionalities, which have led to a 20% reduction in user interaction time, thus enhancing the overall user experience.

Child Vaccination Scheduler

The Child Vaccination Scheduler is a dedicated application designed to facilitate vaccination appointment scheduling for children. The primary objective of this application is to improve immunization rates by as much as 30% when utilized effectively. It includes a comprehensive tracking system for vaccination histories, thereby enhancing compliance with immunization schedules and improving record-keeping practices.

Bike Rental System

The Bike Rental System is a comprehensive platform aimed at optimizing the rental process for users. This project encompasses a robust inventory management system, ensuring 100% availability of vehicles during peak rental periods. By streamlining operations, the system enhances the overall efficiency of the bike rental service.
